deduction game

Rules

  1. 自分の頭にあるカードを当てる推理ゲームです。
  2. 各プレイヤーは手札と頭にカードを持っていて、一部見えないカードがあります。
    1. 自分の手札カードは見えます。
    2. 自分の頭にあるカードは見えません。
    3. 他プレイヤーの手札カードは見えません。
    4. 他プレイヤーの頭にあるカードは見えます。
  3. 他のプレイヤーに質問をして、自分の頭にあるカードを推理していきます。
    1. 自分の手番では質問か宣言を行います。
    2. 質問の答えと宣言結果は全員で共有されます。
    3. 同じ質問を繰り返すことはできません。
